# Screenshots Folder

Place your application screenshots here to display in the main README.

## Recommended Screenshots

1. **dashboard.png** - Main application window
2. **checkin.png** - QR Check-in system in action
3. **clients.png** - Client management interface

## Tips for Screenshots

- Use PNG format for better quality
- Recommended size: 1280x720 or 1920x1080
- Show the application in use with sample data
- Ensure good window size and visibility

## How to Update README

After adding screenshots, they will automatically display in the main README.md file.
